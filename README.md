# Nekobito
Nekobito is a browser-based markdown editor.

## Usage
* Open https://nekobito.github.io
* Write markdown.

## Features
### PWA Support
If you use Google Chrome, you can install Nekobito as a progressive web app.

## Development

### elm version
0.19

### Install node modules
```
yarn
```

### Start dev server


```
yarn run start
```

### Build
```
yarn run build
```
